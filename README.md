# Mouse Latency Tester 🖱️

Mouse Latency Tester is a lightweight browser-based tool designed to measure the responsiveness and performance of your mouse.

It allows users to test how quickly their mouse registers clicks and how consistently they can perform rapid inputs.

This tool can be useful for:

• Testing gaming mouse responsiveness  
• Measuring click speed (CPS – clicks per second)  
• Checking mouse latency between clicks  
• Comparing different mice or settings  
• Evaluating overall clicking performance

## Metrics Measured

The test tracks several important performance metrics:

- **Click Count** – Total number of clicks during the test  
- **Response Time (ms)** – Time between clicks in milliseconds  
- **Fastest Click** – The lowest recorded response time  
- **Average Response Time** – Average click delay across the test  
- **CPS (Clicks Per Second)** – Overall clicking speed

## How the Test Works

The test runs for **10 seconds** and records your click activity in real time.  
At the end of the test, your results are displayed so you can evaluate your mouse performance.

## Use Cases

This tester can help users:

- Benchmark gaming mouse responsiveness  
- Compare new mouse hardware  
- Test clicking speed and consistency  
- Check input performance in browsers
